# BeatForge VST

**Goal:**  
Generate a single MIDI melody (Lead) based on user inputs.

## Inputs
- Tempo (BPM)
- Key (C, D#, etc.)
- Genre (Pop, Trap, EDM)
- Complexity (Simple, Moderate, Complex)

## Output
- A MIDI file containing a melody based on the chosen inputs

## Must-Haves
- Select BPM, Key, and Genre
- Melody generation
- MIDI export

## Nice-to-Haves
- Bass/High layer options
- Text-prompt based melody ideas (AI)
- VST GUI knobs/sliders
